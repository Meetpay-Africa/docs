---
title: 'Get Payment Status'
openapi: 'GET /api/v1/payments/{id}'
---

Returns the current payment including `status`.

## Status values

| Value | Terminal |
| :--- | :--- |
| `pending` | No |
| `processing` (rare) | No |
| `completed` | Yes |
| `failed` | Yes |
| `cancelled` | Yes |
| `expired` | Yes |
| `partially_refunded` | After completion |
| `refunded` | After completion |

PayGrid delivers status transitions to your `webhook_url` as `payment.{status}` events (no webhook at creation, and none for the refund statuses `partially_refunded` / `refunded`). See [Webhooks](/guides/webhooks) and [Payment status](/guides/payments/status).
