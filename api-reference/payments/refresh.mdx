---
title: 'Refresh Payment Status'
openapi: 'POST /api/v1/payments/{id}/refresh'
---

Performs a live status check and updates the payment. Triggers webhooks/callbacks if the status changed.

Use sparingly — each call performs a live upstream check and returns `502 PROVIDER_ERROR` when that check is temporarily unavailable. For routine polling prefer [GET /api/v1/payments/{id}](/api-reference/payments/status); PayGrid also refreshes pending payments automatically.
