## 1. Get your API Keys

To start using MeetPay, you need an active merchant account. Once logged in, you can generate your **Production** and **Sandbox** API keys from the [Dashboard](https://meet.briq.tz).

## 2. Authenticate your requests

All API requests must include your API key in the `Authorization` header.

```bash
Authorization: Bearer <your_api_key>
```

## 3. Make your first request

Verify your connection by fetching your merchant profile and balance.

<CodeGroup>
```bash cURL
curl -X GET "https://meet.briq.tz/api/v1/merchants/me/balance" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```javascript Node.js
const axios = require('axios');

axios.get('https://meet.briq.tz/api/v1/merchants/me/balance', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
})
.then(response => console.log(response.data))
.catch(error => console.error(error));
```
</CodeGroup>

## Supported Networks

MeetPay supports all major mobile money networks in Tanzania:

| Network | Type |
| :--- | :--- |
| **Vodacom** | M-Pesa |
| **Tigo** | Tigo Pesa |
| **Airtel** | Airtel Money |
| **Halotel** | Halopesa |
| **TTCL** | T-Pesa |
